Présentation du projet

LFI3 est une plateforme web permettant l'analyse et l'exploration de contenus textuels multilingues (arabe, français, anglais). Elle s'appuie sur Elasticsearch pour l'indexation et la recherche full-text, et propose des visualisations graphiques des données.

Frontend Angular 13 + Angular Material
Backend PHP 8.1 + Apache
Base de données MySQL 8.0
Recherche Elasticsearch 7.17
Infrastructure Docker + Docker Compose
